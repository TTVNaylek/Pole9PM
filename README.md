# Pole 9 Passwords Manager (P9PM)

# Introduction:
A passwords manager for the Pole 9 association

# Features:
- Administrator can:
    - Add/Remove user account
    - Add/Remove account information (E-mails, passwords)
- Users can:
    - Use account information (E-mails, passwords)
